## рҹ“– Project Overview

MovieVerse is a responsive web application that allows users to search for movies and instantly view detailed information using the **OMDb (Open Movie Database) API**.

The application features a modern user interface, real-time API integration, dark/light theme switching, local storage support, search suggestions, favorites management, keyboard shortcuts, and more to provide an engaging user experience.

# рҹҢҗ API Used

### OMDb API

The application fetches movie information from the Open Movie Database API.

API Website

https://www.omdbapi.com/

Example Request

```
https://www.omdbapi.com/?apikey=YOUR_API_KEY&t=Avatar
```

Example Response

```
{
  "Title":"Avatar",
  "Year":"2009",
  "Genre":"Action, Adventure, Fantasy",
  "imdbRating":"7.9"
}
```
